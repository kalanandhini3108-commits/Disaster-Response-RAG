# Disaster-Response-RAG
A Retrieval-Augmented Generation (RAG) system that provides disaster response guidance using PDF-based knowledge retrieval and local language generation.

## Features

* PDF document ingestion
* Text chunking and embedding generation
* FAISS vector database for retrieval
* Context-aware disaster response generation
* Local inference using Hugging Face models
* No API key required

## Tech Stack

* Python
* LangChain
* FAISS
* Sentence Transformers
* Hugging Face Transformers
* PyPDF

## Workflow

PDF Documents → Chunking → Embeddings → FAISS → Retrieval → Response Generation

## Example Queries

* What should people do during floods?
* How should communities prepare for earthquakes?
* Emergency response guidelines during disasters

This project demonstrates an end-to-end RAG pipeline for disaster preparedness and emergency information assistance.
